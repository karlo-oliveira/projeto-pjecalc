# projeto-pjecalc

Contribuição para a aula na plataforma DIO, sobre utilização do NotebookLM, incluindo meu projeto sobre PJe-Calc Cidadão, o aplicativo da Justiça Federal para a área de perícia financeira e trabalhista.

O objetivo é testar na prática os recursos do NotebookLM, analisando as fontes de PDFs e vídeo aulas, para gerar alguns materiais no Estúdio: apresentação de slides, um post para blog e um mapa mental. 

Todos os materiais gerados têm o mesmo fim: utilizar corretamente o PJe-Calc Cidadão para efetuar os cálculos trabalhistas do FGTS, no serviço de perícia financeira.

Fiquei muito satisfeito com a facilidade e eficácia do NotebookLM, que conseguiu consolidar as informações e filtrar exatamente o que eu desejava, nos prompts. De modo geral, o NotebookLM se mostrou extremamente útil para reforçar o aprendizado e também para facilitar o acesso a quem está buscando as informações pela primeira vez.
